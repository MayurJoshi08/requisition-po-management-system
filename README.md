# Requisition & Purchase Order Management System

A web-based system to manage company requisitions and purchase orders with approval workflow.

## Features
- Create requisition request
- Email approval system
- Approve or reject requisitions
- Generate Purchase Order
- Digital signature support
- PDF generation

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Purpose
This system helps companies manage internal purchase requests and approvals digitally instead of using paper processes.

## Future Improvements
- Multi-level approval workflow
- Dashboard analytics
- Vendor management
